# EXPAND-000: 全国展開 概要・優先順位

## 目標

東京専用の銭湯マップを、全国47都道府県に拡張する。公開APIと各都道府県の銭湯組合公式サイトを組み合わせて高品質なデータを集約する。

## データ戦略

### 一次データ（信頼性最高）
**各都道府県の公式銭湯組合サイト** をスクレイピング。

- 全浴連傘下の47都道府県組合がサイトを持つ
- 公式データなので名前・住所・営業時間・定休日・電話番号の精度が最高
- robots.txt でコンテンツページへの制限なし（調査済み）

### 座標補完（OSM）
組合サイトに緯度経度がない場合、**OSM Overpass API** で補完。

- 完全無料・ODbL ライセンス（商用利用・DB 保存可）
- 日本国内に約5,624件の `amenity=public_bath` データ
- 帰属表示（`© OpenStreetMap contributors`）が必須

### 情報補完（Yahoo! API）オプション
**Yahoo! ローカルサーチAPI** で評価点・写真を付与。

- 無料枠50,000リクエスト/日
- 業種コード `gc=0418` で銭湯・浴場を検索可能
- データ二次配布禁止（表示目的での保存は可）

### 除外したAPI
| API | 除外理由 |
|---|---|
| Google Places API | データの DB 保存・二次配布が利用規約上制限される |
| 楽天トラベル / じゃらん | 宿泊施設専用、銭湯カテゴリなし |
| ぐるなびAPI | 2021年に一般公開終了（法人有償のみ） |
| ホットペッパーAPI | 飲食店専用 |
| 国土地理院API | POI データなし（住所→座標のみ） |

## チケット一覧と実施順序

```
EXPAND-002 (DB拡張)
    ↓
EXPAND-001 (全国スクレイパー) ─── EXPAND-003 (OSM座標補完)
    ↓
EXPAND-005 (フロント全国対応)
    ↓
EXPAND-004 (Yahoo! 評価・写真) ← オプション
```

| チケット | 内容 | 優先度 | 工数感 |
|---|---|---|---|
| **EXPAND-002** | DB スキーマ拡張（prefecture, source_url 追加） | 最高 | 小（半日） |
| **EXPAND-001** | 全国銭湯組合スクレイパー拡張 | 高 | 大（2〜3日） |
| **EXPAND-003** | OSM Overpass API 座標補完 | 高（組合サイトに座標がない場合に必須） | 中（1日） |
| **EXPAND-005** | フロント全国対応（都道府県フィルタ・地図広域化） | 高 | 中（1日） |
| **EXPAND-004** | Yahoo! ローカルサーチAPI で評価・写真補完 | 低（付加価値） | 中（1日） |

## 注意事項

- 各都道府県組合サイトの**利用規約確認**を実施前に行うこと（東京と同様にチケットに記録）
- リクエスト間隔は最低2秒
- 「休業中」「廃業」表記の銭湯はフィルタリングするか、DB にフラグ保持するか設計を決めること（EXPAND-001 で判断）
